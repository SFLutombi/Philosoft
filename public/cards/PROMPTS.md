# PhiloSift Archetype Card Prompt Pack

Use this pack to generate vertical tarot-style cosmic thinker portraits that match the brand.

## Target Export Settings
- Format: WEBP (preferred) or PNG
- Resolution: 1080 x 1920 (9:16)
- Safe center zone: keep face, key symbol, and title glyph inside middle 70% height
- Lighting: dramatic chiaroscuro, warm gold accents, deep charcoal shadows
- Color direction: charcoal black, parchment ivory, muted bronze, ritual gold
- Texture: subtle grain, faint paper wear, archival print finish
- No visible text in image (title text is rendered by app)
- Character direction: androgynous cosmic beings only, no explicit gender coding
- Vibe direction: gritty occult tarot, dark fantasy, philosophical symbolism over literal realism

## Global Style Prefix
Use this prefix at the start of every prompt:
Tarot-inspired cosmic philosopher entity, androgynous non-gendered character design, dark-academia ritual aesthetic, gritty dark fantasy texture, cinematic chiaroscuro, archival grain, elegant composition, symbolic object in frame, premium card illustration, moody but readable, high detail, portrait orientation 9:16.

## Negative Prompt
Use this with each generation:
low resolution, blurry face, extra fingers, duplicate limbs, modern streetwear logos, neon cyberpunk palette, bright purple glow, cartoon style, anime style, flat vector, watermark, text overlay, signature, frame clipping, cropped head, overexposed highlights.

## Recommended Workflow
1. Generate each prompt at 1080 x 1920.
2. Pick the best candidate for each archetype.
3. Upscale lightly only if needed, avoid over-sharpening.
4. Export using the exact filenames listed above.
5. Place files in public/cards.
6. Rebuild and verify each archetype loads the intended artwork.
